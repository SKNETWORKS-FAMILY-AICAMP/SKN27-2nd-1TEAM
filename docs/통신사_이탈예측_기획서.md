# 📊 Project Re:tain | 통신사 고객 이탈 방어 시뮬레이션 시스템 (Draft)

> **고객의 이탈을 예측하고, 마케팅 정책 변화에 따른 위험도 변화를 실시간으로 시뮬레이션하는 의사결정 지원 솔루션**

---

## 1. 프로젝트 개요 (Overview)
본 프로젝트는 통신사 가입자 데이터를 활용하여 **이탈(Churn) 가능성**을 점수화하고, 가상의 리텐션 정책(요금제 변경, 약정 갱신 등)을 적용했을 때의 **위험도 감소 효과를 시뮬레이션**하는 시스템 구축을 목표로 합니다. 단순한 '예측'을 넘어 현업에서 활용 가능한 '전략 수립 도구'로서의 인터페이스를 지향합니다.

---

## 2. 주요 목표 (Core Objectives)
* **High-Precision Prediction**: 통신사 특유의 계약 및 행동 데이터를 기반으로 정교한 이탈 예측 모델 구축.
* **Interactive Dashboard**: 현재 가입자 상태와 이탈 위험군 분포를 시각화하여 상시 모니터링 제공.
* **Policy Simulation**: 특정 고객군을 대상으로 요금 할인, 부가서비스 제공 등 정책 적용 시 위험도 변화 측정.
* **Operational Insight**: 모델의 판단 근거(Feature Importance)를 통해 실질적인 방어 전략 도구 제공.

---

## 3. 데이터셋 설계 (Data Strategy)
통신사 도메인의 특성을 반영하여 피처를 3가지 카테고리로 분류하여 관리합니다.

### 3.1 피처 분류 (Feature Classification)
1. **고정 변수 (Immutable)**: 성별, 연령대, 가입 기간(Tenure).
2. **행동 변수 (Behavioral)**: 월평균 데이터 사용량, 로밍 횟수, 고객센터 상담(Complain) 횟수.
3. **제어 변수 (Actionable - 시뮬레이션 대상)**: 
   - **Contract**: 약정 형태 (Month-to-month, One year, Two year)
   - **MonthlyCharges**: 월 납부 요금
   - **PaperlessBilling / Auto-Renew**: 결제 방식 및 자동 갱신 여부
   - **TechSupport / OnlineSecurity**: 부가서비스 가입 여부

---

## 4. 시스템 아키텍처 (System Architecture)

### 4.1 페이지 구성
1. **[Home] Executive Dashboard**
   - KPI: 전체 가입자 수, 평균 이탈 확률, 고위험군(Risk > 70%) 비중.
   - 시각화: 요금제별 이탈률 비교, 데이터 사용량과 이탈의 상관관계 그래프.
2. **[Model] Engine Insight**
   - 모델 성능 지표 (Recall, F1-Score, ROC-AUC).
   - 변수 중요도 (SHAP / Feature Importance).
3. **[Simulation] Strategic Sandbox (핵심 기능)**
   - 타겟 고객군 필터링 및 정책 파라미터(할인율, 요금제 변경 등) 설정.
   - 적용 전(Original) vs 적용 후(Simulated) 리스크 변화 비교.

---

## 5. 시뮬레이션 기능 상세 (Core Logic)

### 5.1 전체 프로세스
1. **고객군 선택**: (예: 약정 만료 3개월 전 고위험 고객)
2. **파라미터 입력**: 슬라이더/토글을 통해 정책 값 조정 (예: 요금 10% 할인 적용).
3. **가상 데이터 생성**: 선택된 고객군의 `simulated_df` 생성 및 변수 값 변조.
4. **재예측 (Re-prediction)**: 기존 학습된 모델에 변조된 데이터 주입하여 확률값 재산출.
5. **결과 시각화**: 이탈 확률 분포의 좌측 이동(Risk 감소) 현상 리포팅.

### 5.2 시뮬레이션 파라미터 예시
- **Monthly Discount**: 0% ~ 30% 범위의 요금 할인 적용.
- **Contract Renewal**: 무약정 고객의 약정 전환(1/2년) 시뮬레이션.
- **Service Bundle**: 스트리밍 서비스 또는 보안 서비스 무료 탑재 효과.

---

## 6. 결과 시각화 설계 (Visualizing Impact)

### 6.1 상단 요약 UI
- **Target Audience**: 최근 데이터 사용 감소 고객 (845명)
- **Applied Policy**: 요금 15% 할인 + 1년 약정 갱신 유도
- **Simulation Result**: 평균 위험도 12%p 감소 예상

### 6.2 주요 그래프
- **Risk Density Plot**: 정책 적용 전/후의 이탈 확률 밀도 함수 비교 (Before vs After).
- **Segment Migration**: 고위험(High)에서 중/저위험(Mid/Low)으로 이동한 고객 수 집계.

---

## 7. 기대 효과 및 활용 방안
- **마케팅 효율화**: 전체 고객이 아닌 이탈 징후가 확실한 타겟에게만 프로모션 집중.
- **브랜드 충성도 강화**: 불만 고객에게 선제적인 혜택을 제시하여 이탈 방어.
- **데이터 기반 의사결정**: "감"이 아닌 "예측 데이터"에 근거한 리텐션 캠페인 기획.

---

- **Dashboard**: Streamlit
